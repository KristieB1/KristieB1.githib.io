Project: Python weather

The Python project required knowledge from previous exercises to create functions in python which utlised csv formatted weather data to make pre scripted unit tests pass. The functions included: Reformating and converting data, identifying min and max position and creating weather summaries using these functions. The most challenging part of this project was learning to use the datetime library to format iso standard date time into human readable text, i was able to overcome this with a few google searches and mentor help. Overall I found this project challenging but highly rewarding; i have included a sample code snippet and test run results below

Link to project on github: Weather project


            import csv
            from datetime import datetime
            
            DEGREE_SYBMOL = u"N{DEGREE SIGN}C"
            
            def format_temperature(temp):
                """Takes a temperature and returns it in string format with the degrees
                    and celcius symbols.
            
                Args:
                    temp: A string representing a temperature.
                Returns:
                    A string contain the temperature and "degrees celcius."
                """
                return f"{temp}{DEGREE_SYBMOL}"
            
            def convert_date(iso_string):
                """Converts and ISO formatted date into a human readable format.
            
                Args:
                    iso_string: An ISO date string..
                Returns:
                    A date formatted like: Weekday Date Month Year e.g. Tuesday 06 July 2021
                """
                date_string=datetime.strptime(iso_string, "%Y-%m-%dT%H:%M:%S%z")
                return date_string.strftime("%A %d %B %Y")
